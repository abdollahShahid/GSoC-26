## Brief Repository Description

A minimal repository for my GSoC 2026 work on improving FOSSology’s CycloneDX export support, with weekly progress reports and a short project overview.

## Project Summary

This project focuses on improving FOSSology’s existing CycloneDX SBOM export support by upgrading it from CycloneDX 1.4 to 1.7.

The planned work includes:

- updating `specVersion` and `$schema` to CycloneDX 1.7
- preserving the current export workflow through `ui_cyclonedx`
- improving license and compliance field mapping
- enriching supported metadata where reliable evidence exists
- adding validation and regression coverage
